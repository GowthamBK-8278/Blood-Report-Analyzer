🩸 Blood Report Analyzer

📌 Description

This project analyzes patient blood test data to identify abnormal health conditions such as high sugar levels and low hemoglobin. It helps in understanding patient health status using data analysis techniques.



🎯 Objective

- To analyze blood test data
- To detect abnormal values
- To visualize patient health insights



🛠️ Tools Used

- Excel (Data Preparation)
- SQL (Data Analysis)
- Python (Pandas for processing)
- Power BI (Dashboard Visualization)



📊 Features

- Detects high sugar levels
- Identifies low hemoglobin
- Provides clear insights using charts
- Easy-to-understand dashboard



📂 Dataset

The dataset contains patient details such as:

- Name
- Hemoglobin level
- Sugar level
- Cholesterol







🚀 Conclusion

This project helps in quickly identifying health risks and provides useful insights for better decision-making.
